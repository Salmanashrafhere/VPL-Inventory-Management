# VPLAssist+ Inventory Management System

A WPF‑based Inventory Management application developed using C# and WPF.  
This project demonstrates GUI design, object‑oriented programming, file handling, and multithreading.

## Features
- Add, update, delete, and search products
- Displays product list in DataGrid with computed totals
- Inventory saved in JSON format using Newtonsoft.Json
- Responsive UI with background loading and saving

## How to Run
1. Clone this repository
2. Open `VPLAssistPlus.sln` in Visual Studio
3. Restore NuGet packages
4. Build and run the application

## Technologies
- C#
- WPF (.NET Framework 4.7.2)
- JSON Persistence
